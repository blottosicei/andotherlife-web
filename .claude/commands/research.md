# 블로그 키워드 리서치

주제: $ARGUMENTS

## 지시사항

아래 단계에 따라 "$ARGUMENTS" 주제에 대한 SEO 키워드 리서치를 수행하세요.

### 1단계: 키워드 클러스터 분석

`context/target-keywords.md`를 읽고 "$ARGUMENTS"와 관련된 카테고리와 기존 키워드를 파악하세요.

### 2단계: 키워드 확장

주제와 관련된 키워드를 확장하세요:
- **핵심 키워드** (1-2개): 메인 타겟 키워드
- **보조 키워드** (3-5개): 관련 검색어
- **롱테일 키워드** (3-5개): 구체적인 질문형/상황형 검색어
- **관련 질문** (4-6개): FAQ 후보 ("~인가요?", "~할까요?" 형태)

### 3단계: 검색 의도 분류

각 키워드의 검색 의도를 분류하세요:
- 정보형 (Informational): 지식/정보를 찾는 검색
- 상업형 (Commercial): 서비스/상품을 비교하는 검색
- 거래형 (Transactional): 예약/구매를 원하는 검색

### 4단계: 콘텐츠 전략 제안

`context/seo-guidelines.md`와 `context/internal-links-map.md`를 참조하여:
- 추천 카테고리 (7개 중)
- 추천 글 제목 (3개 후보)
- 타겟 독자 (내담자/전문가)
- CTA 전략 (어떤 상담 프로그램/교육으로 연결할지)
- 내부 링크 전략 (연결할 기존 콘텐츠)

### 출력 형식

리서치 결과를 구조화된 마크다운으로 출력하세요. 이 결과는 `/project:write-blog` 커맨드의 입력으로 사용됩니다.
