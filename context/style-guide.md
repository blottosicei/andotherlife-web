# 앤아더라이프 블로그 스타일 가이드

## 문서 구조

### 제목 계층
- **H1:** 글 제목 (반드시 1개만 사용, 주요 타겟 키워드 포함)
- **H2:** 주요 섹션 구분 (5-8개 권장, 키워드 자연스럽게 포함)
- **H3:** H2 하위 세부 항목 (필요 시에만, 남용 금지)
- H4 이하: 사용하지 않음

### H2 개수 기준
| 글 길이 | 권장 H2 수 |
|--------|----------|
| 2,000자 이하 | 4-5개 |
| 2,000-3,000자 | 5-6개 |
| 3,000-4,000자 | 6-8개 |
| 4,000자 이상 | 7-8개 (상한) |

---

## 문단 구성

- 문단당 **3-5문장** (이상적으로는 4문장)
- 문단 사이 **빈 줄 1개**로 구분 (Markdown에서 빈 줄 = 단락 구분)
- 한 문단에 하나의 핵심 아이디어만 담기
- 도입 문단: 독자의 공감을 얻는 상황 묘사 또는 핵심 질문으로 시작
- 마지막 문단: 행동 유도 또는 요약 + 부드러운 CTA

---

## 리스트 사용 규칙

### 불릿 리스트 (`-`)
- 3개 이상 나열할 때 사용
- 순서가 없는 항목 목록에 적합
- 각 항목은 간결하게 (1-2줄 이내)
- 리스트 앞에 간략한 소개 문장 필요

### 번호 리스트 (`1.`)
- 순서가 중요한 단계 설명에 사용
- 절차, 방법, 프로세스 설명 시 적합

### 금지 사항
- 2개 이하 항목에 리스트 사용 (문장으로 쓸 것)
- 중첩 리스트 2단계 초과 금지
- 리스트만으로 본문 대체 금지 (문단과 리스트 혼용)

## 볼드 및 강조

### 볼드 (`**텍스트**`)
- 핵심 키워드 첫 등장 시
- 중요한 통계나 수치
- 독자가 반드시 알아야 할 핵심 정보

### 금지 사항
- 문단 전체 볼드 처리
- 미적 목적의 볼드 (강조 의미 없는 사용)
- 한 문단에 3개 초과 볼드 사용

### 이탤릭 (사용 최소화)
- 심리학 용어나 외래어 첫 등장 시 가능
- 인용문 표시

## 인용 및 참고문헌

### 인용 형식
- 본문 인용: `(저자명, 연도)` 형식
  - 예: `(Beck, 1979)`, `(보건복지부, 2023)`
- 직접 인용: 큰따옴표 사용 후 출처 표기
  - 예: "불안은 미래에 대한 과도한 걱정"(APA, 2013)

### references 배열 구성
글 JSON 데이터의 `references` 필드에 출처를 추가합니다:
```json
{
  "references": [
    {
      "author": "저자명",
      "year": 2023,
      "title": "논문/자료 제목",
      "source": "학술지명 또는 기관명",
      "url": "https://example.com (선택)"
    }
  ]
}
```

### 인용 가능 출처 우선순위
1. 국내외 피어 리뷰 학술지
2. 보건복지부, 국립정신건강센터 등 정부 기관
3. APA, WHO 등 국제 전문 기관
4. 대학 부속 연구소 발행 자료

## FAQ 작성 기준

### 구성
- **개수:** 4-6개 (검색 의도에 맞게 선별)
- **형식:** 자연어 질문 형태

### 질문 작성 패턴
- "~인가요?" (Yes/No 답변이 있는 질문)
- "~할까요?" (방법/절차 질문)
- "~는 무엇인가요?" (정의 질문)
- "~는 얼마나 걸리나요?" (기간/비용 질문)
- "~와 ~의 차이는 무엇인가요?" (비교 질문)

### 답변 작성 기준
- 각 답변: 2-4문장
- 질문에 직접 답하는 문장으로 시작
- 추가 정보나 권유로 마무리
- 각 답변에 키워드 1개 이상 자연스럽게 포함

### FAQ 예시
```
Q: 심리상담은 얼마나 자주 받아야 하나요?
A: 일반적으로 주 1회, 50분 세션으로 시작하는 경우가 많습니다.
   상담 목표와 현재 상태에 따라 격주 또는 월 1회로 조정될 수 있습니다.
   상담사와 함께 본인에게 맞는 주기를 결정하는 것이 가장 좋습니다.
```

## 메타 정보

### 메타 타이틀
- **길이:** 30-60자 (한국어 기준)
- **형식:** 핵심 키워드를 앞쪽에 배치
- **패턴 예시:**
  - `[주요 키워드] - [부가 정보] | 앤아더라이프`
  - `[주요 키워드]: [구체적 내용] | 앤아더라이프`

### 메타 디스크립션
- **길이:** 120-155자 (한국어 기준)
- **포함 요소:**
  1. 주요 키워드 (자연스럽게)
  2. 글의 핵심 내용 요약
  3. 행동 유도 문구로 마무리
- **행동 유도 문구 예시:**
  - "지금 바로 확인해 보세요."
  - "전문가의 시각으로 알아봅니다."
  - "앤아더라이프에서 자세히 살펴보세요."

## 숫자 및 단위 표기

### 숫자
| 기준 | 표기 방식 | 예시 |
|-----|---------|-----|
| 만 단위 이상 | 한글 | 1만 명, 2억 원 |
| 만 단위 미만 | 아라비아 숫자 | 500명, 3회 |
| 퍼센트 | 아라비아 숫자 + % | 30%, 약 15% |
| 순서 | 아라비아 숫자 | 1단계, 2번째 |

### 단위
- 기간: "약 6개월", "평균 8회기"
- 비율: "10명 중 3명", "약 30%"
- 금액: "월 15만 원", "1회 7만 원"

## 외래어 및 전문 용어

### 첫 등장 시 한글(영문) 병기
```
인지행동치료(CBT)
변증법적 행동치료(DBT)
외상 후 스트레스 장애(PTSD)
주의력결핍과잉행동장애(ADHD)
```

### 이후 등장
- 한글 용어만 사용 (영문 반복 불필요)
- 약어는 첫 등장 이후 사용 가능

### 심리학 용어 한국어 표준
- Burnout → 번아웃 (소진으로도 표기 가능)
- Mindfulness → 마음챙김
- Trauma → 트라우마 (외상)
- Resilience → 회복탄력성
- Attachment → 애착

## 도입부 (인트로) 작성 지침

- **길이:** 150-250자
- **구조:**
  1. 독자의 상황/감정에 공감하는 문장
  2. 이 글에서 다루는 내용 소개
  3. 읽으면 얻을 수 있는 것 (독자 이익)
- 주요 키워드를 첫 문단에 자연스럽게 포함
- 질문형 시작 허용 ("번아웃이라는 말을 들어보신 적 있으신가요?")

---

## 마무리 (아웃트로) 작성 지침

- **길이:** 100-200자
- **구조:**
  1. 글 핵심 내용 1-2문장 요약
  2. 독자에 대한 공감 또는 격려
  3. 행동 유도 (상담 예약 또는 관련 글 읽기)
- 상담 예약 링크는 CTA 컴포넌트가 담당하므로 본문에서는 부드럽게 방향 제시
