## 2. 2030 청년상담 상세 페이지 콘텐츠

### /counseling/young-adult 페이지 구조

```
1. 히어로
   - "상담, 부담스럽게 느껴졌다면. 이제는 가볍게 시작해보세요."
   - 첫 상담 무료 / 회당 20,000원

2. 이런 고민이 있다면 (8가지)
   - 직장 스트레스, 진로 고민, 가족 관계, 연인 관계
   - 우울/무기력, 불안/걱정, 자기 이해, 기타

3. 왜 앤아더라이프인가? (차별화)
   - 비용 비교 테이블: 일반 5~10만원 vs 앤아더라이프 2만원
   - 교수급 슈퍼비전 필수
   - 한국상담학회 인증 수련과정
   - 대면/비대면 모두 가능

4. 상담 프로세스 (3단계)
   - 신청서 작성 → 사전 조율 → 정기 상담

5. 가격 안내
   - 첫 상담 무료
   - 회당 20,000원 (50분)
   - 기본 6회기 (연장 가능)
   - 언제든 중단 가능

6. FAQ (10개)
   - 상담사 배정, 비밀보장, 저렴한 이유, 상담 횟수/기간
   - 전체 과정, 큰 문제 없어도 되는지, 병원과 차이
   - 예약 취소/변경, 사회적 인식, 신청 방법

7. CTA
   - "혼자 끙끙 앓지 마세요. 첫 상담은 무료입니다."
   - 상담 신청 버튼 → /contact
```
