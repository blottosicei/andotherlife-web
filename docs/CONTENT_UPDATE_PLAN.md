## 2. 로고 이미지 적용 계획

### 2.1 에셋 준비

현재 프로젝트 루트에 있는 로고 파일을 `public/images/` 디렉토리로 이동:

| 원본 파일 | 이동 경로 | 용도 |
|-----------|-----------|------|
| `logo_text.png` | `public/images/logo-text.png` | 텍스트 로고 (기본 사용) |
| `logo_symbol.png` | `public/images/logo-symbol.png` | 심볼 로고 (파비콘, 소형 아이콘) |

추가 생성 필요:
- `public/favicon.ico` — logo_symbol.png 기반 32x32 파비콘
- `public/icon.svg` — logo_symbol.png 기반 SVG 변환 (또는 PNG 유지)
- `public/apple-touch-icon.png` — logo_symbol.png 기반 180x180
- `public/images/logo-text-white.png` — 어두운 배경용 반전 텍스트 로고 (흰색/밝은색)

### 2.2 로고 적용 위치

#### 텍스트 로고 (logo-text.png) — 밝은 배경

| 위치 | 파일 | 현재 상태 | 변경 내용 |
|------|------|-----------|-----------|
| **헤더 (PC)** | `components/layout/Header.tsx` | `<span>` 텍스트 "앤아더라이프" (font-dangam) | `<Image src="/images/logo-text.png">` 로 교체. 높이 약 36-40px, 자동 비율 |
| **헤더 (모바일)** | `components/layout/MobileMenu.tsx` | `<span>` 텍스트 "앤아더라이프" | 동일하게 `<Image>` 교체 |
| **센터 소개 페이지 상단** | `app/about/page.tsx` | 텍스트만 표시 | 헤드라인 옆 또는 위에 텍스트 로고 배치 (선택사항) |
| **OG 이미지** | `app/layout.tsx` 또는 `opengraph-image` | 현재 미설정 또는 기본 | 로고 포함 OG 이미지 생성 고려 |

#### 텍스트 로고 반전 (logo-text-white.png) — 어두운 배경

| 위치 | 파일 | 현재 상태 | 변경 내용 |
|------|------|-----------|-----------|
| **푸터** | `components/layout/Footer.tsx` | `<span>` 텍스트 "앤아더라이프" (text-white) | 반전(흰색) 텍스트 로고 이미지로 교체 |
| **히어로 섹션** (배경이 어두울 경우) | `components/home/HeroSection.tsx` | 해당 없음 | 어두운 배경 오버레이 사용 시 반전 로고 적용 |

> **참고:** 텍스트 로고는 어두운 회색(#555) 글자. 밝은 배경(#f9f9f6, #fff)에서는 원본 그대로 사용하고, 어두운 배경(#0d0f0d 푸터, 진한 히어로)에서는 CSS `filter: brightness(0) invert(1)` 또는 별도 흰색 버전 사용.

#### 심볼 로고 (logo-symbol.png) — 소형·아이콘

| 위치 | 파일 | 현재 상태 | 변경 내용 |
|------|------|-----------|-----------|
| **파비콘** | `app/layout.tsx` → `icons` | `/icon.svg` 참조 | logo_symbol 기반 favicon.ico, icon.png 생성 후 교체 |
| **Apple Touch Icon** | `app/layout.tsx` | `/apple-touch-icon.png` | logo_symbol 기반 180x180 생성 |
| **모바일 메뉴 닫기 상태 아이콘** | 선택사항 | 없음 | 모바일 헤더 좌측에 작은 심볼 로고 (24x24) 표시 고려 |
| **로딩 스피너/스플래시** | 선택사항 | 없음 | 로딩 UI에 심볼 로고 사용 가능 |
| **이메일 서명** | 선택사항 | 해당없음 | 향후 이메일 템플릿 사용 시 |

### 2.3 로고 컴포넌트 구현 권장

**추가할 상담 철학 텍스트:**
> "'그리고' 다른 삶 '& Other Life'"
>
