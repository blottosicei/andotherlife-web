# Meta 캠페인 브리프: 2030 청년상담

## 캠페인 개요

| 항목 | 내용 |
|------|------|
| 캠페인명 | 2030 청년상담 — 첫 상담 무료 |
| 플랫폼 | Meta (Facebook + Instagram) |
| 랜딩페이지 | https://notherlife.com/counseling/young-adult |
| 시작일 | 2026-04-14 (예정) |
| 운영 방식 | 상시 운영 + 반복 최적화 |

## 캠페인 목표

- **1차 목표**: 랜딩페이지 유입 → 상담 예약 신청 (contact 폼 제출)
- **2차 목표**: 브랜드 인지도 확보 (2030 세대 심리상담 니즈 타깃)

## KPI

| 지표 | 목표값 | 비고 |
|------|--------|------|
| CTR | > 1.5% | 업계 평균 0.9% 대비 |
| CPC | < 1,000원 | 초기 학습 기간 후 목표 |
| CVR (랜딩→신청) | > 3% | contact 폼 제출 기준 |
| CPA (상담 신청) | < 30,000원 | 첫 달 목표 |
| ROAS | 측정 예정 | 상담료 기반 산출 |

## 핵심 USP (Unique Selling Points)

1. **첫 상담 무료** — 진입 장벽 최소화
2. **회당 20,000원** — 일반 상담(10~15만원) 대비 파격적 가격
3. **한국상담학회 수련과정** — 민간자격증 아닌 학회 공인 전문성
4. **교수급 슈퍼비전** — 상담사 혼자 판단하지 않는 안전장치
5. **병원 기록 미등재** — 취업·보험 심사 영향 없음

## 전환 퍼널

```
Meta 광고 노출
  → 클릭 (랜딩페이지 유입)
    → 페이지 스크롤 + 체크리스트 인터랙션
      → CTA 클릭 ("상담 신청하기")
        → /contact 폼 제출 (전환)
```

## 예산 계획

| 기간 | 일 예산 | 월 예산 | 목적 |
|------|---------|---------|------|
| 1~2주 (학습) | 10,000원 | ~150,000원 | 픽셀 학습, 오디언스 테스트 |
| 3~4주 (최적화) | 15,000~20,000원 | ~400,000원 | 성과 기반 스케일업 |
| 2개월 이후 | 데이터 기반 조정 | - | ROAS 기반 예산 배분 |

## 성과 추적

- Meta Pixel: 랜딩페이지 PageView + contact 폼 Lead 이벤트
- UTM 파라미터: 모든 광고 URL에 UTM 필수 적용
- GA4: 유입 소스별 전환율 비교
