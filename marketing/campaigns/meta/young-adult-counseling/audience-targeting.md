# 오디언스 타깃팅 전략: 2030 청년상담

## 핵심 타깃 페르소나

### 페르소나 A — 직장인 번아웃
- **연령**: 25~34세
- **성별**: 전체 (여성 비중 높을 것으로 예상)
- **상태**: 직장 스트레스, 번아웃, 이직 고민
- **고민**: "이 일을 계속해야 하나", "매일 출근이 힘들다"

### 페르소나 B — 진로 불확실 청년
- **연령**: 22~29세
- **성별**: 전체
- **상태**: 취준생, 대학원생, 사회 초년생
- **고민**: "내가 뭘 하고 싶은지 모르겠다", "이 선택이 맞는지"

### 페르소나 C — 감정 조절 어려움
- **연령**: 20~35세
- **성별**: 전체
- **상태**: 우울감, 불안, 대인관계 어려움
- **고민**: "나만 이런 건가", "감정 기복이 너무 심하다"

## Meta 광고 타깃팅 설정

### 오디언스 세트 1 — 관심사 기반 (Broad)

| 설정 | 값 |
|------|-----|
| 위치 | 대한민국 전체 (또는 수도권 집중) |
| 연령 | 22~35세 |
| 성별 | 전체 |
| 상세 타깃팅 — 관심사 | 심리학, 정신건강, 자기계발, 명상, 마음챙김 |
| 상세 타깃팅 — 행동 | - |
| 제외 | 심리상담사, 상담심리학 (전문가 제외) |

### 오디언스 세트 2 — 고민 키워드 기반

| 설정 | 값 |
|------|-----|
| 위치 | 대한민국 |
| 연령 | 20~34세 |
| 관심사 | 스트레스 관리, 우울증, 불안장애, 자존감, 감정 관리 |
| 추가 | 온라인 쇼핑 + 모바일 중심 사용자 |

### 오디언스 세트 3 — Lookalike (향후)

| 설정 | 값 |
|------|-----|
| 소스 | contact 폼 제출자 Custom Audience |
| 유사도 | 1~3% |
| 조건 | 최소 100명 축적 후 생성 |

## 플레이스먼트 전략

| 플레이스먼트 | 우선순위 | 이유 |
|-------------|---------|------|
| Instagram Feed | 높음 | 2030 세대 메인 플랫폼 |
| Instagram Stories | 높음 | 몰입도 높은 풀스크린 |
| Instagram Reels | 중간 | 영상 소재 준비 후 |
| Facebook Feed | 중간 | 25~35세 커버리지 |
| Facebook Stories | 낮음 | 초기에는 제외 |

## 리타깃팅 전략 (Phase 2)

| 오디언스 | 조건 | 메시지 |
|---------|------|--------|
| 랜딩페이지 방문자 | 7일 내 방문, 미전환 | "아직 고민 중이신가요? 첫 상담은 무료입니다" |
| 50% 스크롤 | 페이지 절반 이상 열람 | "관심 가져주셔서 감사합니다. 부담 없이 시작하세요" |
| 체크리스트 인터랙션 | 3개 이상 체크 | "해당되는 항목이 있으셨군요. 상담이 도움이 됩니다" |
